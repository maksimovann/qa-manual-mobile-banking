# QA Manual Testing Project: Mobile Banking

# О проекте

Это учебный проект по ручному тестированию мобильного банковского приложения.

В рамках проекта подготовлены тест-кейсы, чек-листы и баг-репорты для основных пользовательских сценариев мобильного банка.

Проект создан для практики manual QA и оформления QA-портфолио.

# Что проверяется

В проекте тестируются:

- авторизация
- вход по паролю
- переводы между счетами
- переводы по номеру карты
- история операций
- негативные сценарии
- ошибки валидации

# Что есть в репозитории

```text
checklists/       чек-листы
test-cases/       тест-кейсы
bug-reports/      баг-репорты
```

# Какие навыки показывает проект

- функциональное тестирование
- smoke testing
- regression testing
- негативное тестирование
- написание тест-кейсов
- работа с чек-листами
- оформление баг-репортов
- проверка пользовательских сценариев
